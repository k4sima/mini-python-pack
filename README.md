Setup embeddable python and pip

# Install

```
# default 3.9.9

./embed_python_setup.sh <PYTHON_VERSION>
```

# Usage

### Run python command

```
python/python.exe test.py
```

### Run pip command

```
python/python.exe -m pip install flask
```
